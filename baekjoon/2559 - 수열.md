연속합을 구한 다음에 k만큼 차를 두고 빼면 될거 같은데?

```
a    = 3 -2 -4  -9   0  3  7 13  8 -3
psum = 3  1 -3 -12 -12 -9 -2 11 19 16


```

k = 2 일때 

ps(1)  = a0 + a1
ps(2) - ps(0) = a1 + a2
ps(3) - ps(1) = a0+a1+a2+a3 - (a0+a1) = a2 + a3

k = 5일때
ps(4)  = a0+a1+a2+a3+a4
ps(5) - ps(0) = a1+a2+a3+a4+a5
ps(6) - ps(1) =  a2+a3+a4+a5 + a6
...

```
#include <stdio.h>
#define max(a,b) ((a)>(b)?(a):(b))

int a[100000];
int psum[100000];

int main() {
    int n, k, i, m;
    scanf("%d %d", &n, &k);
    for (i=0; i<n; i++) {
        scanf("%d", &a[i]);
    }
    psum[0] = a[0];
    for (i=1; i<n; i++) {
        psum[i] = psum[i-1] + a[i];
    }

    m = psum[k-1];
    for (i=k; i<n; i++) {
        m = max(psum[i] - psum[i-k], m);
    }
    printf("%d\n", m);

    return 0;
}
```
