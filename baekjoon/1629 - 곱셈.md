https://www.acmicpc.net/problem/1629

$a^n$ 구할때 n이 짝수일때와 홀수일때 나눠서..

실제 구현할때는 범위 넘어가는 거만 주의하면 된다.

```
#include <stdio.h>

int p(int a, int b, int c) {
    if (b == 0) {
        return 1;
    }
    if (b == 1) {
        return a % c;
    }

    if (b % 2 == 0) {
        long x = p(a, b/2, c);
        return (x*x)%c;
    } else {
        return ((long)(a%c)*p(a, b-1, c))%c;
    }
}

int main() {
    int a,b,c;
    scanf("%d%d%d", &a,&b,&c);

    printf("%d\n", p(a, b, c));
    return 0;
}

```
