주어진 좌표가 어느 사분면에 위치하는지 파악하여 각 사분면에서의 첫 좌표의 offset을 계산하고, 그 사분면 안에서의 좌표로 조정 한 후 아랫단계의 재귀함수를 호출.

```c
#include <stdio.h>

#define LOG 0
#define write_log(format,args...)        \
                  if (LOG) {                      \
                      printf(format, ## args);    \
                  }


int f(int s, int n, int r, int c) {
    if (n == 1) {
        return s + (r*2 + c);
    }

    int pow2 = 1<<(n-1);
    int q = (r/pow2)*2 + c/pow2;
    int offset = q*pow2*pow2;

    r = r - (r/pow2)*pow2;
    c = c - (c/pow2)*pow2;
    write_log("n = %d, q = %d, r = %d, c = %d, offset = %d\n", n, q, r, c, offset);
    return f(s + offset, n-1, r, c);
}

int main() {
    int n, r, c;
    scanf("%d%d%d", &n, &r, &c);

    printf("%d\n", f(0, n, r, c));

    return 0;
}

```
