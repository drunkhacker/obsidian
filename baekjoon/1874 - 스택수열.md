invariant들을 보면  
1. 스택에 오름차순으로 집어넣기 때문에 스택에 존재하는 숫자 역시 다 오름차순.

그래서 예를들면, $a_i$가 만들어지려면, stack의 top이 $a_i$보다 작아야함. 아니면 더 이상 push를 해봤자 $a_i$보다 큰 놈만 들어갈거. 
top = $a_i$ ->  그냥 뽑으면 되고,
top < $a_i$ -> $a_i$ 가 나올때까지 push 한후 pop

ex) 43687521

초기 stack의 top은 0이라고 가정

$a_i$   top
4 > 0 -> push 1
4 > 1 -> push 2
4 > 2 -> push 3
4 > 3 -> push 4
4 = 4 -> pop -> 4
3 = 3 -> pop -> 3
6 > 3 -> push 5
6 > 5 -> push 6
6 = 6 -> pop -> 6
8 > 5 -> push 7
8 > 7 -> push 8
8 = 8 -> pop -> 8
7 = 7 -> pop -> 7
5 = 5 -> pop -> 5
..

ex2) 1 2 5 3 4 
$a_i$   top
1 > 0 -> push 1
1 = 1 -> pop -> 1
2 > 0 -> push 2
2 = 2 -> pop -> 2
5 > 0 -> push 3
5 > 3 -> push 4
5 > 4 -> push 5
5 = 5 -> pop -> 5
3 < 4 !! -> fail 

대신 fail이 나면 NO만 출력해야하기때문에 스택에 연산을 수행하는 즉시 print를 할 수는 없으므로 어디에다가 따로 저장이 필요하다.

```
#include <stdio.h>

int a[100001] = {0,};
char ops[100001] = {};

struct stack {
    int arr[100001];
    int sp;
} stack;

void push(struct stack* s, int v) {
    s->arr[s->sp++] = v;
}

int pop(struct stack* s) {
    return s->arr[--s->sp];
}

int peak(struct stack* s) {
    return s->arr[s->sp - 1];
}

int main() {
    int n, i, j, x, ok;
    scanf("%d", &n);

    // init stack
    stack.sp = 0;
    push(&stack, 0);

    for (i=0; i<n; i++) {
        scanf("%d", &a[i]);
    }

    i = j = 0;
    x = 1;
    ok = 1;
    while (i<n) {
        if (a[i] < peak(&stack)) {
            ok = 0;
            break;
        }

        if (a[i] > peak(&stack)) {
            push(&stack, x++);
            ops[j++] = '+';
        } else {
            pop(&stack);
            ops[j++] = '-';
            i++;
        }
    }

    if (ok) {
        for (i=0; i<j; i++) {
            printf("%c\n", ops[i]);
        }
    } else {
        printf("NO\n");
    }

    return 0;
}

```
