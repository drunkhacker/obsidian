i번째 포도주를 마신다고 했을때 경우의 수가..

i-1 번째꺼 먹고 i 번째 꺼 먹음
근데 i-2 번째꺼는 못먹었을테니 (3잔 연속 금지)
f(i-3) -> a(i-1) -> a(i)

i-1번째꺼 안먹고 i번째 꺼 먹음
f(i-2) -> a(i)

dp\[i\] = i번째 포도주를 마신다고 했을때 최대 양
dp\[i\] = max(
  dp\[i-3\] + a\[i-1\] + a\[i\],
  dp\[i-2\] + a\[i\]
)

이게 맞는지 모르겠는데.. [[2579 - 계단 오르기]]는 마지막 계단을 무조건 밟는다는 가정이 있으니 i번째껄 먹는다는 식으로 접근이 가능한데, 얘는 중간에 끊을 수 있어서 그게 안될듯.

sub problem을 다음과 같이 정의.
dp\[i\] = i 번째 포도주를 마지막으로 먹었다고 했을때 최대 포도주의 양.
그리고 나면 우리가 구하는 답은 max(dp1, .. dpn) 이 될것임. 

dp\[i\] = max(dp\ii-3\] + ai-1 + ai, dp\ii-2\] + ai)

초기 조건
dp(0) = a0
dp(1) = a0 + a1
dp(2) = max(a1+a2, a0+a2)
