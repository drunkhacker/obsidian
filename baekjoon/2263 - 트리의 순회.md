https://www.acmicpc.net/problem/2263

inorder 순서를 a1 a2 a3  .. an 라 하고 
post order 순서를 b1 b2 b3 .. bn 라 하면

bn은 이 트리의 root다. 포스트 오더는 left child, right child, node 순이기 때문에,  루트 노드가 맨 뒤에 나타난다.
bn을 {an}에서 찾았을때 ai라고 하면, ai 기준으로 왼쪽, 오른쪽 subtree로 나눌 수 있다. 
(a1... ai-1) ai (ai+1 ... an) 이에 대응되는 b 배열은
(b1... bi-1)(bi...bn-1) bn 이고, 

이렇게 되면 a1 ... ai-1, b1 ... bi-1 및 ai+1 ... an, bi ... bn-1 두 문제로 쪼개지고 같은 방식으로 풀면 된다.

종료조건은 순열의 길이가 3이하일때. 
a1 a2 a3는 
```
  a2
  / \
a1  a3
```
의 트리다. 

a1 a2만 가지고는 a1이 left child인지 (a2가 root인지), a1이 root인지 (a2가 right child 인지) 알수가 없다.
b1 b2 를 같이 봤을때, b2가 root이므로, b2가 a1과 같은지 a2 와 같은지 보고 결정.
b2 == a2 라면
```
  a2
  / \  
a1   x
```
b2 == a1 라면
```
  a1
  / \
 x  a2  
```

a1만 있으면 a1이 단일 루트이고 자식이 없는 트리 (leaf)

```c
#include <stdio.h>
#include <stdlib.h>

#define LOG 0
#define write_log(format,args...)        \
                  if (LOG) {                      \
                      printf(format, ## args);    \
                  }


int n;
int as[100000];
int bs[100000];

typedef struct tree Tree;
typedef struct tree {
    int data;
    Tree *left;
    Tree *right;
} Tree;

Tree *newnode(int elem) {
    Tree *t = malloc(sizeof(Tree));
    t->data = elem;
    t->left = NULL;
    t->right = NULL;
    return t;
}

Tree *build(int *a, int *b, int len) {
    write_log("build, len = %d\n", len);
    if (len == 0) {
        return NULL;
    }
    if (len == 1) {
        return newnode(a[0]);
    }

    int root = b[len-1];
    write_log("root(from b) = %d\n", root);
    // find root in a 
    int i;
    for (i=0; i<len; i++) {
        if (a[i] == root) {
            write_log("root index = %d, root = %d\n", i, root);
            break;
        }
    }

    Tree *t = newnode(root);
    t->left = build(a, b, i);
    t->right = build(a+i+1, b+i, len-i-1);

    return t;
}

void preorder(Tree *t) {
    if (t == NULL) {
        return; 
    }
    printf("%d ", t->data);
    preorder(t->left);
    preorder(t->right);
}


int main() {
    int i;
    scanf("%d", &n);
    for (i=0; i<n; i++) { 
        scanf("%d", &as[i]);
    }
    for (i=0; i<n; i++) { 
        scanf("%d", &bs[i]);
    }

    Tree *root = build(as, bs, n);
    preorder(root);
    putchar(10);

    return 0;
}

```
