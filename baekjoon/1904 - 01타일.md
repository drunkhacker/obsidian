https://www.acmicpc.net/problem/1904

크기가 N이면, 

00\[N-2\]로 만드는 수 
1\[N-1\] 로 만드는 수

올라가면서 만들자 
N\[1\] = 1 (1)
N\[2\] = 2 (00, 11)
N\[3\] = N\[1\] + N\[2\] = 3
.. 피보나치인가?

재귀로 풀면 스택 터지는듯 
```
#include <stdio.h>

int m[1000001] = {0,};


int main() {
    int n, i;
    scanf("%d", &n);
    m[1] = 1;
    m[2] = 2;
    for (i=3; i<=n; i++) {
        m[i] = (m[i-1] + m[i-2])%15746;
    }

    printf("%d\n",m[n]);

    return 0;
}
```
