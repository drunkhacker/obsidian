https://www.acmicpc.net/problem/5639

preorder 표현이 `a1 a2 a3 ... an` 라면 `a1`은 root이다. a2 ... an에서 left child와 right child를 가르려면, 

이진 검색 트리의 특성상 left child는 모두 a1보다 작고, right child는 모두 a1보다 크므로, ai(i≥2) 중 처음으로 a1보다 커지는 i = x를 찾으면 `a1 | a2 ... a[x-1] | ax ... an` 으로 가를 수 있다.

이 다음부터는 문제는 재귀적으로 풀 수 있다.

트리를 직접 다 구성할 필요는 없고, preorder인 root - left child - right child를 left child - right child - root 로 출력하면 되므로, `convert` 함수는 
```
convert(tree) :=
    if (tree.length == 0) then 
	    return
    if (tree.length == 1) then 
	    print(tree) 
		return
		
	let [left, right, root] := tree
	convert(left)
	convert(right)
	print(root)
```
처럼 구성할 수 있다.


```c
#include <stdio.h>

int a[10000];
int n;

void convert(int *elems, int size) {
    if (size == 0) {
        return;
    }

    if (size == 1) {
        printf("%d\n", elems[0]);
        return;
    }

    int root = elems[0];
    int i;
    for (i=1; i<size; i++) {
        if (elems[i] > root) {
            break;
        }
    }

    // [1..i-1]=left, [i..size-1]=right

    convert(elems+1, i-1);
    convert(elems+i, size-i);
    printf("%d\n", root);
}

int main() {

    while (fscanf(stdin, "%d", &a[n++]) != EOF) { } 
    n--; // as it contains EOF as last elem
    convert(a, n);
    return 0;
}
```
