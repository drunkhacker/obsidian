연산자 우선순위 없이 계산된다 했으므로.. 
연산자 입력을 array로 세팅하고 (ops)
`ops[n]` 을 for 문 돌리면서 ..? -> n번째 depth에 쓸 연산자 고르는거.. 근데 이게 ops array변형 시키는 식으로 가야하니 사실 좀 짜증나고 
[[15649 - N과 M (1)]]와 똑같이 visit array두면 되고, 비슷한 방식으로 접근하면 될듯

```
fun backtrack(value, n) { // value = 지금까지 계산된 value
  if (n == N) {
    max < value ? max = value
    min > value ? min = value
  }
  for i in (1...N)
    if (!visit[i]) {
	    visit[i] = true
	    value <- value *ops[i]* a[n] 
	    backtrack(value, n+1)
	    visit[i] = false
    }
}
```

백트래킹 돌릴때 a[0]가 아니라 a[1] 부터 돌려야함. 첫번째 수는 연산 안거치고 바로 계산되기 때문. 그리고 n = 1부터 시작해야함. 왜냐면 다음 타깃은 a[0]가 아니라 a[1]이니까..

```
#include <stdio.h>

int N;
int a[12];
int ops[12];
int v[12] = {0, };

int min = 2147483647;
int max = -2147483648;

#define MAX(a, b) ((a)>(b) ? (a):(b))
#define MIN(a, b) ((a)<(b) ? (a):(b))

int cal(int op, int a, int b) {
    switch(op) {
        case 0: return a+b;
        case 1: return a-b;
        case 2: return a*b;
        case 3: return a/b;
    }
    return 0;
}

void backtrack(int value, int n) {
    int i;
    if (n == N) {
        max = MAX(max, value);
        min = MIN(min, value);
        return;
    }

    for (i=0; i<N-1; i++) {
        if (!v[i]) {
            v[i] = 1;
            backtrack(cal(ops[i], value, a[n]), n+1);
            v[i] = 0;
        }
    }
}

int main() {
    int i, j, n;
    scanf("%d", &N);
    for (i=0; i<N; i++) {
        scanf("%d", &a[i]);
    }

    int op = 0; // 0,1,2,3 : + - * /
    int* cop = &ops[0];
    for (i=0; i<4; i++) {
        scanf("%d", &n);
        for (j=0; j<n; j++) {
            *cop++ = op;
        }
        op++;
    }

    backtrack(a[0], 1);

    printf("%d\n%d\n", max, min);
    return 0;
}
```
