i번째 계단에 왔다면 경우의 수는 다음과 같다.

- i-2번째에서 2계단 올라옴 . (i-2) -> i
- i-1번째에서 1계단 올라옴
	- 근데 세 계단 연속으로 못가니까 다음과 같은 경우밖에 없음
	- (i-3) -> (i-1) -> i

dp\[i\] 를 i번째 계단을 밟았을때 최대 합이라고 하면 
dp\[i\] = max( dp\[i-2\] + a\[i\] , dp\[i-3\] + a\[i-1\] + a\[i\] )

```
#include <stdio.h>
#define max(a,b) ((a)>(b)?(a):(b))

int dp[300] = {0, };
int a[300] = {0, };

int main() {
    int i, n;

    scanf("%d", &n);
    for (i=0; i<n; i++) {
        scanf("%d", &a[i]);
    }

    dp[0] = a[0];
    dp[1] = a[0] + a[1];
    dp[2] = max(a[0] + a[2], a[1] + a[2]);
    for (i=3; i<n; i++) {
        dp[i] = max(dp[i-2] + a[i], dp[i-3] + a[i-1] + a[i]);
    }

    printf("%d\n", dp[n-1]);

    return 0;
}
```
