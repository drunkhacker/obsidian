[[1463 - 1로 만들기]]와 비슷한 문제인데, 실제로 연산이 적용되는 과정을 출력하라는 문제.

[[1463 - 1로 만들기]]에서 `dp[i]`는 `i`를 1로 만들기 위한 연산 횟수의 최소값...
`dp[i] = j`라 하면 그 다음에는 `dp[j] = k`를 따라가고 그 다음에는 `dp[k]`를 따라가고.. 결국엔 1이 될때까지 반복.

예를들면 10의 경우, 
`dp[10]`을 계산할 때, 
`dp[9] + 1` , `dp[5] + 1` 중 작은 값을 가져 오게 돼있고, 10의 다음 연산의 결과값은 9혹은 5이다.

따라가는 배열을 `b[]`라고 하면 `b[i] = j` 일때 연속적으로 `b[j]`를 탐색해가면서 `b[j]`가 바닥까지 올때까지 따라가게 한다. `dp[]`계산할때 `b[]`도 같이 결정하면 됨.

```c
#include <stdio.h>

#define LOG 0
#define write_log(format,args...)        \
                  if (LOG) {                      \
                      printf(format, ## args);    \
                  }



int dp[1000001];
int b[1000001];

int main() {
    int n;
    scanf("%d", &n);

    dp[1] = 0; b[1] = 0;
    dp[2] = 1; b[2] = 1;

    for (int i=3; i<=n; i++) {
        dp[i] = dp[i-1] + 1;
        b[i] = i-1;
        write_log("b[%d] = b[%d] = %d\n", i, i-1, b[i]);
        if (i%3 == 0) {
            if (dp[i] > dp[i/3] + 1) {
                dp[i] = dp[i/3] + 1;
                b[i] = i/3;
                write_log("b[%d] = b[%d] = %d\n", i, i/3, b[i]);
            }
        }

        if (i%2 == 0) {
            if (dp[i] > dp[i/2] + 1) {
                dp[i] = dp[i/2] + 1;
                b[i] = i/2;
                write_log("b[%d] = b[%d] = %d\n", i, i/2, b[i]);
            }
        }
    }

    printf("%d\n", dp[n]);
    while (n != 0) {
        printf("%d ", n);
        n = b[n];
        write_log("next n = %d\n", n);
    }
    printf("\n");

    return 0;
}

```
