규칙을 찾아보면, 각 kring 안의 셀 갯수는 레벨별로,
1, 6, 12, 18, 24 개.. 식으로 나가므로, 각 레벨별 마지막 셀 인덱스는 
Lv1 : 1
Lv2: 2 ~ 7
Lv3 : 8 ~ 19
Lv4 : 20 ~ 37 ..

Lv n : Lv n 에 속하는 셀 번호로 도착하려면 n개의 셀을 지나야 한다.

로부터, $a_n = a_{n-1} + 6(n-1)$ 로 뽑아낼 수 있다.

```c
#include <stdio.h>

int main() {
    int n;
    int a = 1;
    scanf("%d", &n);

    for (int i=1;;i++) {
        if (n <= a) {
            printf("%d\n", i);
            break;
        }
        a = a + 6*i;
    }

    return 0;
}

```
