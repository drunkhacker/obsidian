https://www.acmicpc.net/problem/1717

union find 문제 . disjoint set을 표현하는데, 각 원소의 representative가 누군지 저장하는 parent 배열을 이용한다.

- x,y 집합을 합치는것은 y의 repr의 parent을 x의 repr로 대치하는것.
- x,y가 같은 집합인지 보는것은 x의 repr와 y의 repr가 같은지 보는것.
- repr을 찾는 함수를 `find()`라 하면,
```
find(x) = 
	if parent[x] = x then return x
	else 
		r = find(parent[x])
		parent[x] = r // path compression
		return r
```
으로 표현할 수 있음.

```c
#include <stdio.h>

#define N 1000000

int parent[N+1];

int find(int x) {
    if (parent[x] == x) {
        return x;
    }

    int r = find(parent[x]);
    parent[x] = r;
    return r;
}

void merge(int a, int b) {
    int ra = find(a);
    int rb = find(b);

    parent[rb] = ra;
}

int main() {
    int i, n, m;
    for (i=0; i<=N; i++) {
        parent[i] = i;
    }
    scanf("%d %d", &n, &m);
    int x, a, b;
    for (int i=0; i<m; i++) {
        scanf("%d %d %d", &x, &a, &b);
        if (x == 0) {
            merge(a, b);
        } else { 
            printf("%s\n", find(a) == find(b) ? "YES" : "NO");
        }
    }
    return 0;
}

```
