파라메트릭 서치인데, 무한루프에 안빠지게 mid 를 조절 잘 할것
overflow 조심할것.

```c
#include <stdio.h>
#include <stdlib.h>

int a[10000];

int comp(const void *a, const void *b) {
    return (*(int *)a) > (*(int *)b) ? 1 : (*(int *)a) < (*(int *)b) ? -1 : 0;
}

int solve(int n, int k) {
    int st, en, i, mid;
    long c;

    en = a[k-1];
    st = 1;

    while (st < en) {
        mid = ((long)st + en + 1)/2;

        for (c=0, i=0; i<k; i++) {
            c += (a[i] / mid);
        }

        if (c >= n) {
            st = mid;
        } else {
            en = mid - 1;
        }
    }

    return st;
}

int main() {
    int n, k, i;

    scanf("%d %d", &k, &n);
    for(i=0; i<k; i++) {
        scanf("%d", &a[i]);
    }

    qsort(a, k, sizeof(int), comp);

    printf("%d\n", solve(n, k));

    return 0;
}
```
