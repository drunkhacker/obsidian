dp\[i\] = a\[i\]로 최장 증가 부분 수열이 끝날때의 그 수열의 길이
ai가 부분수열의 마지막 요소가 되려면, ai가 가장 큰 값이어야 즉, 그 부분수열의 마지막 값이 ai보다 작아야지.
즉, ai를 마지막으로 가지는 가장 긴 부분수열의 길이는 ai를 추가할 수 있는 부분수열 중 가장 길이가 긴 친구 .
ai를 추가할 수 있는 부분 수열은 어떻게 알아? -> i>j 인 j중 ai>aj 인 친구들을 다 보면 되지.

```
#include <stdio.h>
#define max(a,b) ((a)>(b)?(a):(b))

int a[1000];
int dp[1000];

int main() {
    int n, i, j, m, ans;

    scanf("%d", &n);
    for (i=0; i<n; i++) {
        scanf("%d", &a[i]);
    }

    ans = dp[0] = 1;

    for (i=1; i<n; i++) {
        m = 0;
        for (j=0; j<i; j++) {
            if (a[i] > a[j]) {
                m = max(dp[j], m);
            }
        }
        dp[i] = m + 1;
        ans = max(ans, dp[i]);
    }

    printf("%d\n", ans);

    return 0;
}
```
